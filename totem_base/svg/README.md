SVG File Examples
=================

This folder provides a couple examples of SVG's that are good for the totem cube panel inserts that provide the silhouette on each of the four sides of the cubes. The SVG format is ideal as it is scalable and the vector graphics provide clean edges that are easy to import into Fusion 360 and cut holes into the panel using the extrude tool. Note that we can convert simple png or jpg’s to svg if you need.

The following should be kept in mind for your panel images:
- There must not be any floating holes (i.e. holes within holes). These won't have any material to support the inner hole. Attaching some bridging material can be added, much like a stencil, can be added if needed.
- Avoid any section thinner than 0.2mm in width. Parts this thin will likely fall apart when printed.
- The more space on your images, the more light will get through from the cube. This is a good thing.
- A round or square image is best to maximize the image on the cube. This isn't essential as we will center it.
- If you don't want your rune centered on the tile, let us know!
